## v0.5.0 (2026-06-17)

### 新功能

**Stack 布局模式**
- book 模板新增 `stack` 布局：顶栏全宽展示站点品牌，下方左侧导航、右侧正文，适合文档类站点
- 用 `nav_file` 替代 `summary`，未配置时自动隐藏侧栏与上下页导航
- Sidebar / TopHeader 按布局模式智能切换站点 Header 与菜单按钮展示

**页面打印与二维码**
- 新增 `layout.hide_print_button` 与 `layout.hide_page_qrcode` 配置项，可按页面或全局控制打印按钮和页面二维码的显示
- front matter 优先于 layout 全局配置，粒度更细

**目录与标题显示控制**
- 新增 `meta.hide_toc`、`meta.hide_title` 控制单页目录与标题的显隐

**AJAX 无刷新导航（PJAX v5）**
- 全面重构为零感知 PJAX 引擎，页面切换无闪烁、无白屏
- 新增链接预取缓存，鼠标悬停即后台加载，导航几乎零延迟
- 布局指纹机制：head 资源变化时自动整页刷新，保证样式与脚本一致性
- 侧栏导航树、TOC、代码高亮等组件在 AJAX 导航后自动重新挂载

**统一锚点滚动**
- 抽取通用锚点滚动工具，支持嵌套滚动容器定位
- TOC 点击、heading 锚点、URL hash 导航行为完全统一
- stack 布局下自动联动顶栏高度，滚动定位准确

### 改进

**侧栏导航树**
- 锚点导航（点击 TOC / 带 hash 的链接）后自动刷新侧栏高亮，不再需要手动滚动到对应位置
- 目录路径 `index.html` 与尾斜杠 `/` 正确等价识别，避免目录首页误高亮
- 节点交互样式下沉至可点击链接，热区更准确；展开/折叠图标垂直居中，多行标题不再错位

**侧栏宽度拖拽**
- 修复 PJAX 导航后侧栏宽度拖拽器失效的问题（浏览器前进/后退、站内链接切换后均可正常拖拽）

**Markdown 排版**
- 移除 h2 左侧装饰条，标题样式更简洁
- 修复仅含 `<code>` 的链接下划线穿透背景的问题
- 标题锚点图标替换为链环样式

**Hash 导航与历史回退**
- 修复浏览器前进/后退（popstate）时误触发全页请求的问题
- 目录（TOC）链接添加 `data-no-ajax`，避免点击目录条目被 pjax 拦截

### 内部优化

- 构建系统命名统一：`ssr-*` 改为 `jsrender-*`，`fe-*` 改为 `client-*`，消除与运行时 SSR 的歧义
- `scrollAnchor` 工具迁入 `youlog_lib/core`，TOC 与 page-ajax 共用，减少重复代码
- widget 文档规范建立，各模块入口聚合到 `index.ts`，README 结构统一
- `set_latest_release` 发布脚本强制校验 changelog，确保每次发布都有版本说明
